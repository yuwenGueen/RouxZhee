# 📝 基础语法

本文档介绍RouxZhee中支持的 Markdown 基础语法特性。

> 💡 **提示**：本文档中的所有语法示例均已在RouxZhee中测试通过，可直接复制使用。

---

## 🎯 基础语法

### 1️⃣ 标题

**语法**：
```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果**：
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

---

### 2️⃣ 段落

**语法**：
```markdown
这是一个普通段落。

这是另一个段落，通过空行分隔。
```

**效果**：
这是一个普通段落。

这是另一个段落，通过空行分隔。

---

### 3️⃣ 强调

**语法**：
```markdown
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___
```

**效果**：
*斜体文本*
_斜体文本_
**粗体文本**
__粗体文本__
***粗斜体文本***
___粗斜体文本___

---

### 4️⃣ 列表

#### 无序列表

**语法**：
```markdown
- 项目一
- 项目二
- 项目三
  - 子项目一
  - 子项目二
```

**效果**：
- 项目一
- 项目二
- 项目三
  - 子项目一
  - 子项目二

#### 有序列表

**语法**：
```markdown
1. 第一项
2. 第二项
3. 第三项
   1. 子项1
   2. 子项2
```

**效果**：
1. 第一项
2. 第二项
3. 第三项
   1. 子项1
   2. 子项2

---

### 5️⃣ 引用

**语法**：
```markdown
> 这是一个引用
> 
> 引用可以有多行
> 
> > 嵌套引用
```

**效果**：
> 这是一个引用
> 
> 引用可以有多行
> 
> > 嵌套引用

---

### 6️⃣ 行内代码

**语法**：
```markdown
使用 `npm install` 安装依赖

在 JavaScript 中，`const` 用于声明常量
```

**效果**：
使用 `npm install` 安装依赖

在 JavaScript 中，`const` 用于声明常量

---

### 7️⃣ 分隔线

**语法**：
```markdown
---
***
___
```

**效果**：
---
***
___

---

### 8️⃣ 脚注

**语法**：
```markdown
这是一个带有脚注的文本[^1]。

[^1]: 这是脚注的内容
```

**效果**：
这是一个带有脚注的文本[^1]。

[^1]: 这是脚注的内容

## 💡 最佳实践

### 1️⃣ 标题层级

- 一级标题 (`#`) 用于文档主标题
- 二级标题 (`##`) 用于主要章节
- 三级标题 (`###`) 用于子章节
- 避免使用过多层级，建议不超过 4 级

### 2️⃣ 段落与换行

- 使用空行分隔段落
- 段落内换行使用两个空格 + 回车

### 3️⃣ 列表使用

- 保持列表项格式一致
- 嵌套列表使用 2 个空格缩进

### 4️⃣ 强调使用

- 使用 `**粗体**` 突出重要内容
- 使用 `*斜体*` 表示强调或外来语
- 避免过度使用，保持文档可读性

### 5️⃣ 文档结构

- 使用一致的标题层级
- 添加适当的空白行分隔内容
- 使用列表和引用使内容更结构化
- 为长篇文档添加目录（RouxZhee会自动生成）
