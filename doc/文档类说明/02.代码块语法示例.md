## 💡 最佳实践

### 1️⃣ 始终指定语言

- 始终为代码块指定语言，以获得正确的语法高亮
- 如果不确定语言，可以使用 `text` 或不指定

### 2️⃣ 保持代码缩进一致

```javascript
// 好的示例 - 缩进一致
function example() {
  if (condition) {
    doSomething();
  }
}

// 不好的示例 - 缩进混乱
function example() {
if (condition) {
   doSomething();
    }
}
```

### 3️⃣ 添加注释说明

对于长代码，考虑添加注释说明：

```python
# 计算斐波那契数列的前n项
def fibonacci(n):
    """
    生成斐波那契数列
    
    参数:
        n: 需要生成的项数
    
    返回:
        包含斐波那契数列的列表
    """
    if n <= 0:
        return []
    
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    
    return fib
```
