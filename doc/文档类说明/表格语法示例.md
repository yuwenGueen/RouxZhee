# 📊 表格语法示例

本文档详细介绍RouxZhee中支持的所有表格语法特性，包含完整的语法示例和效果展示

> 💡 **提示**：本文档中的所有语法示例均已在RouxZhee中测试通过，可直接复制使用

---

## 🎯 基础表格

### 1️⃣ 简单表格

**语法**：
```markdown
| 表头1 | 表头2 | 表头3 |
|------|------|------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |
```

**效果**：
| 表头1 | 表头2 | 表头3 |
|------|------|------|
| 内容1 | 内容2 | 内容3 |
| 内容4 | 内容5 | 内容6 |

---

## 🎨 表格对齐方式

### 2️⃣ 左对齐（默认）

**语法**：
```markdown
| 左对齐 | 左对齐 |
| :----- | :----- |
| 内容1 | 内容2 |
```

**效果**：
| 左对齐 | 左对齐 |
| :----- | :----- |
| 内容1 | 内容2 |

---

### 3️⃣ 居中对齐

**语法**：
```markdown
| 居中对齐 | 居中对齐 |
| :------: | :------: |
| 内容1 | 内容2 |
```

**效果**：
| 居中对齐 | 居中对齐 |
| :------: | :------: |
| 内容1 | 内容2 |

---

### 4️⃣ 右对齐

**语法**：
```markdown
| 右对齐 | 右对齐 |
| -----: | -----: |
| 内容1 | 内容2 |
```

**效果**：
| 右对齐 | 右对齐 |
| -----: | -----: |
| 内容1 | 内容2 |

---

### 5️⃣ 混合对齐方式

**语法**：
```markdown
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容1 | 内容2 | 内容3 |
```

**效果**：
| 左对齐 | 居中对齐 | 右对齐 |
| :----- | :------: | -----: |
| 内容1 | 内容2 | 内容3 |

---

## 🌟 高级表格技巧

### 6️⃣ 表格中包含链接

**语法**：
```markdown
| 名称 | 链接 |
| :--- | :--- |
| Vue.js | [官网](https://vuejs.org) |
```

**效果**：
| 名称 | 链接 |
| :--- | :--- |
| Vue.js | [官网](https://vuejs.org) |

---

### 7️⃣ 表格中包含代码

**语法**：
```markdown
| 命令 | 示例 |
| :--- | :--- |
| npm install | `npm install vue` |
```

**效果**：
| 命令 | 示例 |
| :--- | :--- |
| npm install | `npm install vue` |

---

### 8️⃣ 表格中包含强调文本

**语法**：
```markdown
| 功能 | 优先级 |
| :--- | :--- |
| 用户登录 | **高** |
```

**效果**：
| 功能 | 优先级 |
| :--- | :--- |
| 用户登录 | **高** |

---

## 💡 最佳实践

### 1️⃣ 保持表格简洁

- 避免创建过于复杂的表格
- 列数建议不超过 5-7 列

### 2️⃣ 使用对齐方式提高可读性

```markdown
| 文本列 | 数值列 | 日期列 |
| :----- | -----: | :---: |
| 左对齐 | 右对齐 | 居中 |
```

### 3️⃣ 添加表头说明

清晰的表头能让表格更易理解：

```markdown
| 字段名 | 数据类型 | 必填 |
| :----- | :------: | :--: |
```
