# 你好，我的伙计！

欢迎来到 **RouxZhee** 博客框架！🎉

## 特性介绍

这是一个基于 Astro + Vue + TypeScript + SCSS 构建的现代博客框架。

### 主要功能

- ⚡ **极速性能** - 基于 Astro 的 Islands 架构
- 🎨 **优雅设计** - 玻璃质感圆角卡片风格
- 🌙 **主题切换** - 支持明暗主题切换
- 📱 **响应式** - 完美适配各种设备

## 代码示例

```typescript
// 欢迎使用 RouxZhee！
const blog = new RouxZhee({
  theme: 'elegant',
  author: 'your-name'
});

blog.start();
```

### 后续读取 MD 文件的方式

- 直接在 `src/pages` 目录下创建 MD 文件，每个文件对应一个页面。
- 可以使用 Astro 的 `import` 语句在其他 MD 文件中引入当前文件的内容。

```typescript
// 读取 doc 目录下所有 MD 文件
const posts = import.meta.glob('../doc/**/*.md');
```
